# Ứng Dụng Quản Lý và Phân Tích Đồ Thị

Ứng dụng Python để quản lý, phân tích và trực quan hóa đồ thị với các chức năng cơ bản và nâng cao.

## Cấu trúc dự án

```
App_ctrr/
├── src/
│   ├── core/
│   ├── algorithms/
│   ├── visualization/
│   └── utils/
├── tests/
├── data/
├── main.py
├── requirements.txt
└── README.md
```

## Cài đặt

```bash
pip install -r requirements.txt
```

## Chạy ứng dụng

```bash
python main.py
```

## Chức năng

### Phần cơ bản
1. Vẽ đồ thị trực quan
2. Lưu đồ thị
3. Tìm đường đi ngắn nhất
4. Duyệt đồ thị (BFS & DFS)
5. Kiểm tra đồ thị 2 phía
6. Chuyển đổi giữa các phương pháp biểu diễn

### Phần nâng cao
7. Trực quan hóa thuật toán:
   - Prim
   - Kruskal
   - Ford-Fulkerson
   - Fleury
   - Hierholzer
